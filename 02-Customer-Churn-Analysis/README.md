# Customer Churn Analysis

## Project Overview
This projet analyzes cusstomer churn data form a telecommunication company to identify key factors that influence customer attribution.The goal is to derive actionable insights that can support customer retention strategies.

## Dataset
- Source: IBM Telco Customer Churn Dataaet
- File: "WA_Fn-UseC_-Telco-Customer-Churn.csv"
- Records: ~7,000 customers
- Target Variable: "Churn" (Yes / No)

## Tools and Technologies
- Python 3.10
- Matplotlib
- Seaborn
- Jupyter Notebook (VS Code)

## Key Analysis Steps
- Data loading and inspection.
- Data cleaning (handling missing and incorrect valuse).
- Exploratory data analysis (EDA).
- Visual analysis of churn drivers.
- Business-oriented interspection of results.

## Key Insights
- Customers on month-to-month contracts show significantly higher churn ratea.
- Higher monthly charges are associated with increased churn.
- Customers with shorter tenure are more likely to churn.
- Long-term contracts are effective in retaining cusotomers.

## Conclusion
This project demonstrates how customer data can be analyzed to uncover churn patterns and retention risks.The findings can help businesses prioritize retention efforts and improve long-term customer value.
